PyMunin APC PHP Cache Plugin

PyMunin Multigraph Munin Plugin for monitoring APC PHP Cache.

APC PHP Plugin - phpapcstats

The multigraph plugin implements the following graphs:

  • php_apc_memory
  • php_apc_items
  • php_apc_reqs_filecache
  • php_apc_reqs_usercache
  • php_apc_expunge

The PHP script apcinfo.php must be placed somewhere in the document root and have access permissions from localhost to permit retrieval of stats from APC. The script can be found int the external utilities folder (./ext) folder of PyMunin distribution.

Environment Variables

Variable Description
host Web Server Host.
(Default: 127.0.0.1)
port Web Server Port.
(Default: 80, SSL: 443)
user User in case authentication is required for access to APC Status page.
password Password in case authentication is required for access to APC Status page.
monpath APC status script path relative to Document Root.
ssl Use SSL if yes.
(Default: no)
include_graphs Comma separated list of enabled graphs.
(All graphs enabled by default.)
exclude_graphs Comma separated list of disabled graphs.
