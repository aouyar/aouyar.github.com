PyMunin Sangoma Wanpipe Telephony Interfaces Plugin

PyMunin Multigraph Munin Plugin for monitoring Sangoma Wanpipe Digital Telephony Interfaces.

Wanpipe Plugin - wanpipestats

The multigraph plugin implements the following graphs:

  • wanpipe_traffic
  • wanpipe_errors
  • wanpipe_pri_errors_
  • wanpipe_pri_rxlevel

Environment Variables

Variable Description
include_ifaces Comma separated list of wanpipe interfaces to include in graphs.
(All Wanpipe Interfaces are monitored by default.)
exclude_ifaces Comma separated list of wanpipe interfaces to exclude from graphs.
include_graphs Comma separated list of enabled graphs.
(All graphs enabled by default.)
exclude_graphs Comma separated list of disabled graphs.
