PyMunin Memcached Plugin

PyMunin Multigraph Munin Plugin for monitoring Memcached Cache Server.

Memcached Plugin - memcachedstats

The multigraph plugin implements the following graphs:

  • memcached_connections
  • memcached_items
  • memcached_memory
  • memcached_traffic
  • memcached_connrate
  • memcached_reqrate
  • memcached_statget
  • memcached_statset
  • memcached_statdel
  • memcached_statcas
  • memcached_statincrdecr
  • memcached_statevict
  • memcached_statauth
  • memcached_hitpct

Environment Variables

Variable Description
host Memcached Server IP.
(127.0.0.1 by default.)
port Memcached Server Port
(11211 by default.)
include_graphs Comma separated list of enabled graphs.
(All graphs enabled by default.)
exclude_graphs Comma separated list of disabled graphs.
