PyMunin FreeSWITCH Plugin

PyMunin Multigraph Munin Plugin for monitoring FreeSWITCH Soft Switch.

FreeSWITCH Plugin - fsstats

The multigraph plugin implements the following graphs:

  • fs_calls
  • fs_channels

Environment Variables

Variable Description
fshost FreeSWITCH Server
(Default: 127.0.0.1)
fsport FreeSWITCH Event Socket Port
(Default: 8021)
fspass FreeSWITCH Event Socket Password
include_graphs Comma separated list of enabled graphs.
(All graphs enabled by default.)
exclude_graphs Comma separated list of disabled graphs.
