PyMunin MySQL Plugin

PyMunin Multigraph Munin Plugin for monitoring MySQL Database Server.

MySQL Plugin - mysqlstats

The multigraph plugin implements the following graphs:

  • mysql_connections
  • mysql_traffic
  • mysql_slowqueries
  • mysql_rowmodifications
  • mysql_rowreads
  • mysql_tablelocks
  • mysql_threads
  • mysql_proc_status
  • mysql_proc_db
  • mysql_commits_rollbacks
  • mysql_qcache_memory
  • mysql_qcache_hits
  • mysql_qcache_prunes
  • mysql_myisam_key_buffer_util
  • mysql_myisam_key_read_reqs
  • mysql_innodb_buffer_pool_util
  • mysql_innodb_buffer_pool_activity
  • mysql_innodb_buffer_pool_read_reqs
  • mysql_innodb_row_ops

Environment Variables

Variable Description
host MySQL Server IP.
(Defaults to UNIX socket if not provided.)
port MySQL Server Port
(3306 by default.)
database MySQL Database
user Database User Name
password Database User Password
include_engine Comma separated list of MySQL Storage Engines to include in graphs.
(All engines are enabled by default.)
exclude_engine Comma separated list of MySQL Storage Engines to exclude from graphs.
include_graphs Comma separated list of enabled graphs.
(All graphs enabled by default.)
exclude_graphs Comma separated list of disabled graphs.
